Editor Controls

Guide

  • Pan Map: Tahan Space + drag klik kiri.
  • Zoom: Scroll mouse wheel.
  • Place Object: Klik object, lalu klik map (one-shot).
  • Add Text: Klik Add Text, klik map, isi text (one-shot).
  • Move Object/Text: Klik kiri tahan lalu drag.
  • Resize: Pilih object/text lalu tarik titik sudut.
  • Delete: Pilih item lalu tekan Delete, atau double klik kiri item.
  • Realtime Share Internet: Host Map menghasilkan kode room. Cukup bagikan kode itu, user lain bisa Join dari jaringan internet manapun.
